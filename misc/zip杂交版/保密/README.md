## 题目描述

### 解题过程
1. 第一层伪加密，前面后面的加密标识位都改为零
2. 第二层 tar -zxvf 解压(新生会tar吗🤔)
3. 第三层文件头缺失，修补504B0304
4. 第四层分卷解压，得到一个word文档(高级一点的解压缩软件似乎自动支持分卷解压)
5. 第五层docx改后缀zip，也当作压缩包打开，找到flag
